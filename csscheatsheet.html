CSS Cheat Sheet

Font Properties

Font-family
Changes the font family of certain words, sentences, paragraphs


Font-style
Changes text: normal, oblique, italics


Font-weight
Used to specify the weight of the font


Font-size
Used to modify the size of the font


Font
Used to combine all properties of fonts


Color and Background Properties

Color
Changes the color of the text


Background-color
Set the background color of an element


Background-image
Sets the background image of an element


Background-repeat
Determines how a specified background image is repeated


Background
Used to combines all properties of background


Text Properties

Word-spacing
Defines an additional amount of space between words


Letter-spacing
Defines an additional amount of space between characters


Text-align
Used to justify text left/center/right/justify


text-indent
Used to specify the amount of indentation prior to the first line of text


Line-height
Used to control the spacing between baselines of text


Box Properties

margin-top/right/bottom/left
Sets the top/right/bottom/left margin of an element by specifying a length or a percentage


margin
Sets the margins of an element by specifying top/bottom/left/right margins


padding-top/right/bottom/left
Describes the amount of space between the top/right/bottom/left border and the content of the selector


padding
Shorthand for the padding-top/bottom/right/left properties


border-width
Used to set width of an element's border


border-color
Used to set the color of a border


border-style
Sets style of a border - none, dotted, solid, dashed, double


border
Sets width, style, and color of a border


width
Each block-level or replaced element can be given a width, specified as a length, percentage, or as auto


height
Each block-level or replaced element can be given a height, specified as a length, percentage, or as auto


float
Allows text to wrap around an element - left, right, none


clear
specifies whether an element allows floating elements to its sides
