HTML Cheat Sheet

Basic Tags

HTML + /HTML
Creates an HTML doc


head + /head
Sets off the title & other info that isn't displayed


body + /body
Sets off the visible portion of the doc


title + /title
Puts name of the doc in the title bar; when bookmarking pages, this is what is bookmarked


Text Tags

h1 + /h1 -> h6 + /h6
Creates headlines, h1 is largest and h6 is smallest


b + /b
Creates bold text; can also use strong + /strong


i + /i
Creates italic text; can also use em + /em


Links

a href="URL" + text + /a
Creates hyperlink to a URL


a href="mailto:emailaddress + text + /a
Creates hyperlink to an email address


a name = "NAME"
Creates target location within a doc


a href = "#NAME"
Creates link to that target location


Lists

ul + /ul
Creates unordered list


ol start=? + /ol
Creates ordered list -> start from xx, where xx is a counting number


li + /li
Encompasses each list item


Images

img src="URL"/>
Adds image


Forms

form + /form
Defines a form


option
Sets off each menu item


textarea name=? cols="x" rows="y" + /textarea
Creates a text box area. Columns set width, rows set height


input type="radio" name=? value=?
Creates radio button


input type="text" name=? size=?
Creates one-line text area. Size sets length in characters


input type="submit" value=?
Creates submit button. Value sets text in the button


input type="reset"
Creates reset button


Tables

table + /table
Creates a table


tr + /tr
Sets off each row in table


td + /td
Sets off each cell in a row


th + /th
Sets off table header -> normal cell with bold, centered text


Table Attributes


td rowspan=?
Sets number of rows a cell should span


td colspan=?
Sets number of columns a cell should span
